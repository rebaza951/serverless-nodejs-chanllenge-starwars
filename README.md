# Nodejs-serverless-challenge
Stack: Nodejs, DynamoDb, Serverless framework, Aws lambda functions

### Install via npm:
```
npm install
```
### Run locally:
```
npm start
```
### Deploy
```
serverless deploy -v
```
### Undeploy
```
serverless remove
```
